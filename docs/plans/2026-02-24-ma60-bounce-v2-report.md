# MA60 支撑反弹策略 v2 — 回测报告

> 回测区间：2025-02-12 ~ 2026-02-13 | 初始资金：100 万

---

## 一、策略概述

在 MA60 均线处于上升趋势时，捕捉股价短暂跌破 MA60 后强力反弹的信号，快速入场、快速止盈。

---

## 二、选股逻辑（三因子 AND 组合）

### 因子 1：MA60 支撑反弹（ma60_bounce_volume）

| 条件 | 说明 |
|------|------|
| 前 5 天内至少有 1 天收盘价在 MA60 上方 | 确认是"刚跌破"而非长期破位 |
| 前一天收盘价 < MA60 | 确认已跌破支撑 |
| 当天收盘价 > MA60 | 当天重新站上支撑 |
| 当天涨幅 > 5% | 反弹力度足够强 |
| 当天成交量 > 前一天 2 倍 | 放量确认 |

### 因子 2：MA60 近期上升趋势（ma60_recent_uptrend）

| 条件 | 说明 |
|------|------|
| 最近 10 天 MA60 严格递增 | 确保处于上升趋势，不做震荡股 |
| 10 天 MA60 涨幅 ≥ 0.5% | 趋势有实质性斜率 |

### 因子 3：信号质量过滤（signal_quality_filter）

| 条件 | 说明 |
|------|------|
| 跌破 MA60 天数 ≤ 5 天 | 短期跌破才有反弹价值 |
| 20 日波动率 ≤ 20 | 排除极端波动股 |
| 当日成交量 ≥ 5 日均量 × 2 | 相对近期明显放量 |
| 换手率 5% ~ 12% | 排除冷门股和过度投机股 |

---

## 三、交易规则

| 项目 | 规则 |
|------|------|
| **入场** | 信号出现后等待最多 5 天，遇到阴线（收盘 < 开盘）当天收盘价买入 |
| **止盈** | 收益达到 **+10%** 立即卖出 |
| **最大持仓** | 持有满 **15 天**未触发止盈则卖出 |
| **止损** | 不设止损（回测验证 5%/8% 止损均降低收益） |
| **仓位** | 等资金分配，当天有 N 个入场信号则每只分配 现金/N |

## 六、策略特征分析

### 盈利交易特征
- 43 笔盈利中有 **24 笔在 15 天内触发 10% 止盈**，平均 7.3 天出场
- 触发止盈的交易平均收益 **+13.4%**（因部分在 15 天到期前已超过 10%）
- 最快 2 天止盈（300907、002431、002824）

### 亏损交易特征
- 43 笔亏损中绝大多数持满 15 天到期卖出
- 平均亏损 -5.4%，最大单笔亏损 -13.04%
- 亏损笔数虽与盈利相当，但**平均亏损幅度远小于平均盈利幅度**（盈亏比 1.67）

### 不设止损的原因
| 止损设置 | 总收益率 | 胜率 |
|---------|---------|------|
| 无止损 | **+105.10%** | 50.0% |
| 8% 止损 | +34.28% | 52.9% |
| 5% 止损 | +2.83% | 46.0% |

该信号类型波动较大（反弹信号本身就是急涨急跌），过早止损会被频繁"洗出"后错过反弹。15 天最大持仓已起到自然止损的作用。

## 七、风险提示

1. 回测区间仅 1 年（2025.02 ~ 2026.02），样本量有限
2. 未考虑交易手续费和滑点（实际收益会略低）
3. 未考虑涨跌停限制（部分交易可能无法按收盘价成交）
4. 策略在单边下跌市场中可能表现不佳（11 月连续亏损段回撤 -16.85%）
5. 过去表现不代表未来收益
